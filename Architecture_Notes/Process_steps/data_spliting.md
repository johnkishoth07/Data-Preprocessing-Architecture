# ✂️ Data Splitting

## Overview

Data Splitting divides the dataset into training, validation, and testing subsets for fair model evaluation.

## Typical Split

- Training Set (70–80%)
- Validation Set (10–15%)
- Test Set (10–15%)

## Benefits

- Prevent overfitting
- Evaluate model performance
- Ensure unbiased testing

## Outcome

Properly separated datasets for model development.
