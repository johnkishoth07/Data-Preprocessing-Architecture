# 📥 Data Collection

## Overview

Data Collection is the first step in the Machine Learning pipeline. It involves gathering relevant, accurate, and high-quality data from various sources to build a reliable dataset for analysis and model training.

## Common Data Sources

- Databases (SQL, NoSQL)
- CSV Files
- Excel Spreadsheets
- APIs
- Sensors and IoT Devices
- Log Files
- Web Scraping

## Objectives

- Gather relevant data
- Ensure data quality
- Collect sufficient samples
- Keep data up-to-date
- Maintain consistency

## Best Practices

- Use trusted data sources.
- Validate collected data.
- Remove duplicate records.
- Document data sources.
- Protect sensitive information.

## Outcome

A complete and reliable raw dataset ready for data understanding.
