# 🧹 Data Cleaning

## Overview

Data Cleaning improves dataset quality by removing errors, inconsistencies, and incomplete records that may negatively impact model performance.

## Tasks

- Handle missing values
- Remove duplicate records
- Correct inconsistent data
- Fix formatting issues
- Correct spelling and typing errors

## Common Techniques

- Mean/Median/Mode Imputation
- Forward Fill
- Backward Fill
- Remove Null Values
- Drop Duplicates

## Best Practices

- Preserve valuable information.
- Avoid unnecessary data removal.
- Validate cleaned data.

## Outcome

A clean and consistent dataset ready for preprocessing.
