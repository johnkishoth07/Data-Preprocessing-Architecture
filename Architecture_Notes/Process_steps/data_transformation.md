# ⚙️ Data Transformation

## Overview

Data Transformation converts raw data into a machine-readable format suitable for machine learning algorithms.

## Tasks

- Normalize data
- Standardize features
- Encode categorical variables
- Feature scaling
- Binning
- Aggregation

## Common Methods

- Min-Max Scaling
- StandardScaler
- One-Hot Encoding
- Label Encoding

## Outcome

Numerical and standardized data suitable for model training.
